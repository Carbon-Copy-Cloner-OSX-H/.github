## Overview

<a href="#">Carbon Copy Cloner</a> creates complete, bootable backups of Mac drives that restore both data and system functionality rather than just files. Unlike file-based backup solutions, CCC clones produce copies of the entire drive that can boot a Mac directly — if the original drive fails, the backup drive works immediately as a replacement without a reinstallation or restoration process.

The <a href="#">cloning engine</a> copies every file, folder, and system component from the source to the destination drive. The result is a bit-for-bit functional duplicate of the original. <a href="#">Incremental updates</a> on subsequent runs copy only files that have changed since the previous backup, making recurring backups fast and efficient after the initial full clone.

<a href="#">Scheduled backup tasks</a> automate the backup process without manual intervention. Configure tasks to run hourly, daily, weekly, or on a custom schedule. CCC monitors for the destination drive's availability and runs tasks when the backup destination is connected. <a href="#">Task notifications</a> report success or failure via notification center, email, or Pushover alerts, confirming that backups are completing as configured.

The <a href="#">SafetyNet feature</a> protects against accidental data loss during backup operations. Before overwriting files on the destination, CCC moves the previous version to a SafetyNet folder rather than deleting it immediately. This safety layer means deleted or modified files remain recoverable from the SafetyNet archive for a configurable period. <a href="#">Pruning controls</a> manage SafetyNet folder growth by removing older versions automatically based on age or total size limits.

## Key Features

- <a href="#">Bootable disk cloning</a> creates exact functional copies that run the Mac directly if the original drive fails
- <a href="#">Incremental backup updates</a> copy only changed files after the initial clone for fast recurring backups
- <a href="#">Scheduled task automation</a> runs backups hourly daily or weekly when the destination drive is available
- <a href="#">SafetyNet file protection</a> preserves previous file versions before overwriting during each backup run
- <a href="#">APFS snapshot support</a> creates point-in-time recovery points without full duplicate storage overhead
- <a href="#">Individual file restore</a> retrieves specific items from backup destinations without full volume restoration
- <a href="#">Backup content filtering</a> excludes caches temporary files and unwanted patterns from every backup task
- <a href="#">SMART disk health monitoring</a> warns about destination drive hardware problems before failures occur
- <a href="#">Complete migration tool</a> transfers everything from an old Mac to a new one using full disk cloning
